# Bluestock Mutual Fund Analytics Capstone

## Project Overview

This project analyzes mutual fund performance, investor behavior, SIP trends, portfolio allocation, and market benchmarks using Python, SQL, SQLite, Power BI, and financial analytics techniques.

## Objectives

* Clean and validate mutual fund datasets
* Build a SQLite analytical database
* Perform exploratory data analysis
* Calculate performance metrics (CAGR, Sharpe, Sortino, Alpha, Beta, Drawdown)
* Build an interactive Power BI dashboard
* Develop advanced analytics including VaR, CVaR, HHI, cohort analysis, and SIP continuity analysis

## Tech Stack

* Python

* NumPy
* Matplotlib
* Seaborn
